COMMENT CHANGER LE MOT DE PASSE
────────────────────────────────
1. Ouvrez index.html dans votre navigateur.
2. Ouvrez la console développeur (F12 ou Ctrl+Maj+I, onglet "Console").
3. Tapez :
       await Security.generateCredential('votre_nouveau_mot_de_passe')
4. La console affiche une ligne du type :
       a1b2c3...:d4e5f6...
5. Copiez cette valeur dans src/js/security.js, à la ligne :
       const ADMIN_CREDENTIAL = '...';
6. Sauvegardez le fichier et rechargez la page.
7. Supprimez ce fichier (NOUVEAU_MOT_DE_PASSE_ADMIN.txt) du projet.

Le mot de passe n'est jamais stocké en clair : seul un hash PBKDF2
(600 000 itérations, sel aléatoire) est conservé dans le code source.
